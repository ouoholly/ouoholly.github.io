進階版一（add 粗體樣式）

python

解釋：
\033[1m start bold
\033[0m end bold

進階版二（add 顏色樣式）

python

格式

\033[(NUMBER)(NUMBER);(NUMBER)(NUMBER)m
第一個數字：3 = 設置文字顏色 ； 4 = 設置背景顏色
第二個數字：顏色碼

顏色碼

0 black
1 red
2 green
3 yellow
4 blue
5 magenta
6 cyan
7 white
9 default

例子

\033[43m 黃底
\033[41m 紅底
\033[43;31m 黃底紅字
