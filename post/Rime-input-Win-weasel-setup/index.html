前言

最近重裝了小狼毫，記錄一下過程。
話說，Rime 真是我用過 (無論是在 Windows 或 Mac) 最好用的中文輸入法！！
打句子不用選字，能自動識別出合適的字，令打字速度快了十倍，真是超贊！！
